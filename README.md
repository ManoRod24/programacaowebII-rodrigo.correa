## DOM
(modelo de objeto de documento)

# o que faz?
Permite que o javascript converse com o navegador a altere o que vocês veem.

# O que é Manipulação em DOM em Javascript?
Altera dados, cores, fontes, tamanhos e afins na tela e não no código fonte. O código fonte permanece o mesmo, intacto. É uma forma de mudamos o conteúdo sem precisar recarregar a página.

# Por que usar?
Para tornar as páginas mais dinâmicas e interativas. O navegador redesenha apenas aquela parte da tela que é para alterar.
Exemplo: cor do botão ao ser clicado.
